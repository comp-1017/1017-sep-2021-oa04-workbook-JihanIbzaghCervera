table Demo

A table is a way of creating a structured grid display of data. the tags we will be interested in are: que este bien escrito, video tables 1h,22min

  • <table> : identifies the table
  • <thead> : identifies the header row of the table
  • <th> : identifies the text in the head of the column
  • <tbody> : identifies the body of the table (detail rows)
  • <tr> : identifies the start of a row
  • <td> : identifies the data cell on the body row
  • <tfoot> : identifies the footer row of the table

all tags are paired tags (openings and closing tag) The border parameter on the table tag will allow to especify a border around your table. The colspan parameter and rowspan parameter are used on the <td> tab allows you to merger columns on a single row and columns cells across rows.
